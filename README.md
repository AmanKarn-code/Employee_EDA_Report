# 🚀 Enterprise Workforce Analytics & Data Engineering Project



## Project Overview

- **Built** an end-to-end enterprise workforce analytics pipeline using **Python, Pandas, NumPy, Matplotlib, and Seaborn**.
- **Transformed** raw HR and payroll records into a clean, analytics-ready business intelligence dataset for workforce optimization and executive reporting.

---

# ⚙️ Data Engineering Pipeline

## Data Cleaning & Quality Engineering

- **Validated** dataset integrity across **1,020 employee records**.
- **Removed** analytical inconsistencies using structured preprocessing workflows.
- **Detected** and verified **0 duplicate employee records**.
- **Standardized** column formatting using string normalization and whitespace trimming.
- **Converted** corrupted negative phone numbers using **absolute value transformation (`abs()`)**.
- **Casted** salary, age, and operational fields into business-ready datatypes.
- **Split** composite `Department_Region` fields into separate analytical dimensions.
- **Engineered** readable `Full_Name` attributes for executive reporting usability.

## Missing Value Resolution

- **Imputed** missing `Age` values using **median replacement** for outlier-resistant demographic stability.
- **Median Age Applied:** **30 Years**.
- **Imputed** missing `Salary` values using **department-wise median salaries**.
- **Preserved** compensation realism by avoiding global-average salary distortion.

## Feature Engineering

- **Created** regional workforce segmentation dimensions.
- **Enabled** department-level payroll aggregation.
- **Prepared** workforce status metrics for operational analysis.
- **Optimized** dataset structure for EDA visualization pipelines.
